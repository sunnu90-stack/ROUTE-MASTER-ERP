# Route Master ERP OTP & AI Setup

## A. SMS OTP Auto-Read Setup

### Android
1. Build and install the Android app on a physical device.
2. The app now uses `sms_autofill` and obtains the app hash automatically.
3. Use this SMS format for automatic OTP detection:

```
Your Route Master OTP is 123456.
@routemaster.app #<APP_HASH>
```

4. Replace `<APP_HASH>` with the value shown in the login screen under "Android SMS auto-read setup." This app hash is generated by `SmsAutoFill().getAppSignature()`.

5. Add the number and domain to your SMS provider settings if required.

### iOS
- On iOS, use the one-time code format and ensure automatic code suggestions are enabled in Settings.
- Example SMS:

```
Your Route Master OTP is 123456.
```

### Notes
- Auto-read requires native mobile; it will not work on Flutter web.
- If you see a browser reCAPTCHA page, you are on web or your Firebase Phone Auth is not configured for native mobile.
- Use Firebase test phone numbers for development to avoid real SMS and reCAPTCHA.

## B. AI Diagnostics & Live Setup

### AI keys
- Set your Gemini API key in `AppConfig.geminiApiKey`.
- Set your OpenRouter API key in `AppConfig.openRouterApiKey`.

### How the app chooses AI
- If Gemini key is provided, the app tries Gemini first.
- If Gemini fails, it falls back to OpenRouter when a valid key is present.
- If both fail or keys are missing, the app uses a local heuristic fallback.

### Diagnostics
- Open the AI chat screen and tap "Show details" under AI Diagnostics.
- The app displays the last AI request status, including whether Gemini or OpenRouter was reached and any error condition.

### Common issues
- `403` means the API key is invalid or not enabled.
- `404` means the model or endpoint is not found.
- `429` means rate limiting.
- On mobile, ensure internet access and correct API key values.

## C. Web OTP and reCAPTCHA

- Firebase Phone Auth on web requires reCAPTCHA and cannot be bypassed safely.
- For a direct OTP experience, use the app on Android or iOS and set up native app verification.
- For development on web, use Firebase test phone numbers configured in the Firebase Console.
